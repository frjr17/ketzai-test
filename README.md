## ğŸ› ï¸ Getting Started

### Prerequisites

- Node.js 14+
- npm 6+

### Install

```bash
git clone https://github.com/frjr17/ketzai-test.git
cd ketzai-test
npm install
````

### Run Dev Server

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

### Build for Production

```bash
npm run build
```

### Start Production

```bash
npm start
```
